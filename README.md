# Data Science Roadmap

A self-structured, week-by-week path from Beginner to Specialization in Data Science — built and maintained by [Basel Hamdy](https://github.com/BaselHamdy1).

## Stages
- **Beginner** — core programming & data handling (Python, OOP, NumPy, Pandas, Git)
- **Intermediate A** — analysis & tooling (Visualization, SQL, Web Scraping, APIs, Time Series)
- **Intermediate B** — math foundations & ML (Feature Engineering, Math for ML, ML)
- **Advanced** — *TBD*
- **Specialization** — *TBD*

## How to Use
Each week lists **Topics → Resources → Task**. Follow the weeks in order within each stage; complete the linked resource(s), then do the task before moving on.
